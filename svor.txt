1:
cd - færir sig um directory
ls - sýnir innihald skrána sem notandin er í
pwd - sýnir staðinn sem notandin er í
mkdir - býr til nýja skrá

2:
git clone: afritar eða klónar repository
git status: sýnir upplýsingar um ástand og breytingar í skjalinu sem notandin er í 
git diff: þegar sett í gang þá keyrir það diff stillingu á git

3:
git log: þú nærð í kommitin 
git checkout: tekur commit og býr til annað eintak af því í trénu
git branch: það er bendill sem bendir á það sem þú gerðir seinast?

4:
version control er kerfi sem heldur upplýsingar um breytingar í skjölunum í tölvuni og leyfa mörgu fólki að vinna í því á samatíma

5: 
git er gott til að geyma og vinna með skjölin þín á tölvunni

6:
eins oft og það þarf eða eins og í þessu tilviki 7 skipti
